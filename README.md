# 📝 To-Do List

A modern, responsive, and user-friendly **To-Do List Web Application** built using **React**, **TypeScript**, and **Vite**. This project enables users to efficiently manage their daily tasks through a clean interface with features like adding, editing, deleting, completing, and undoing tasks.

🌐 **Live Demo:** https://to-do-list-2-chi-flax.vercel.app/

📂 **GitHub Repository:** https://github.com/RajAstha1/TO-DO-LIST-2

---

## 🚀 Features

- ✅ Add New Tasks
- ✏️ Edit Existing Tasks
- 🗑️ Delete Tasks
- ✔️ Mark Tasks as Completed
- ↩️ Undo Deleted Tasks
- 📱 Fully Responsive Design
- ⚡ Fast Performance with Vite
- 🎨 Clean and Minimal User Interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/RajAstha1/TO-DO-LIST-2.git
```

### Navigate to Project Directory

```bash
cd TO-DO-LIST-2
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
